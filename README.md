# artifact
 EEG Artifact Annotation on the Browser

> **Note:** Uses [simplepwa](https://github.com/nikkifurls/simplepwa) as a template.

### What is this?
**Artifact** is a Progressive Web App (PWA) for annotating EEG artifacts on the browser.
